Docker compose is a tool built on top of Docker engine to run multi-container apps.

Docker compose is by default shipped with the Docker Desktop that we install to run Dockker. The command to find out the version of Docker compose is:
```console
docker-compose --version
```

### A few useful commands
We may already be aware that to list all the images, we need to run:
```console
docker image ls
```

To only list the IDs of all the images, we may run:
```console
docker image ls -q
```

To remove a Docker image from a machine, we can run:
```console
docker image rm <docker-image-id>
```

To remove all images from a machine, we can run:
```console
docker image rm $()
```
